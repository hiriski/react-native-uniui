# react-native-uniui

UI Component Library for React Native. Build on top of the react-native-unistyles

## Installation

```sh
npm install react-native-uniui
```

## Usage


```js
import { multiply } from 'react-native-uniui';

// ...

const result = await multiply(3, 7);
```
